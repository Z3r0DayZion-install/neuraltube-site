# neuraltube-site

Public landing page for [NeuralTube](https://z3r0dayzion-install.github.io/neuraltube-site/).

## What this is

This repository hosts the static public landing page for NeuralTube — a local-first desktop environment for intentional video review.

The site is deployed via GitHub Pages from the root of the `main` branch.

## What this is not

This repository does not contain application source code, release tooling, proof internals, or the frozen release artifact. The private NeuralTube application repository remains separate.

## Release references

Release information shown on this site is grounded in the private release process:

- Frozen baseline commit: `82585128`
- Published release tag: `neuraltube-proof-go-2026-04-15`

These references are factual and reflect the approved release baseline. They are not marketing claims.
